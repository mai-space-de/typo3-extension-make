## Scaffolding Commands

* `make:controller` — Extbase controller with standard actions
* `make:event` — PSR-14 event + listener stubs
* `make:exception` — typed exception class
* `make:migration` — database migration class
* `make:trait` — PHP trait stub
* `make:viewhelper` — Fluid ViewHelper stub
* `make:service` — service class with optional DI configuration
* `make:middleware` — PSR-15 middleware stub
* `make:upgrade-wizard` — TYPO3 upgrade wizard stub

## Developer Tools

* TYPO3 Console — extended CLI via `helhum/typo3-console` (cache flush, database compare, etc.)
* T3 Editor — in-backend TypoScript/PHP code editor via `cms-t3editor`
