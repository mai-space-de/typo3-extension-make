============
Maispace Make
============

:Extension key:
    maispace_make

:Package name:
    maispace/make

:Version:
    1.0.0

:Language:
    en

:Description:
    Extended TYPO3 make/kickstarter CLI tool. Provides ``bin/typo3 make:*`` commands to
    scaffold controllers, events, event listeners, exceptions, migrations, middlewares,
    services, traits, upgrade wizards, and ViewHelpers.
